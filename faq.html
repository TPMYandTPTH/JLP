応募・面接について

未経験でも応募できますか？ +
はい、未経験の方も歓迎です。入社後に研修があり、必要なスキルはそこで身につけられます。大切なのは、コミュニケーションが好きで新しい環境に挑戦する意欲です。
英語力はどの程度必要ですか？ +
日本語中心のポジションは日常会話レベルでも応募可能です。社内連絡で英語を使う場面もありますが、GoFluent等の語学プログラムで入社後に学習できます。英語バイリンガル向けのポジションもあります。
面接の回数・方法を教えてください。 +
通常はオンラインで2回（採用担当→マネージャー）です。各30〜45分、Microsoft Teamsを使用。進行は日本語が中心です。
内定までの目安期間は？ +
書類選考（約3営業日）→面接2回→最終面接結果（約1週間）で、全体2〜3週間が目安です。
カジュアル面談は面接ですか？ +
いいえ、選考ではありません。約30分で仕事・生活・移住の疑問を日本語で相談できます。未経験でもOK。

ビザ・移住について

就労ビザの取得は難しい？どれくらいかかる？ +
専門チームが日本語でサポートします。必要書類の案内から申請まで一貫対応。内定から1〜2ヶ月で発行が目安。申請費用は会社負担です。
渡航・到着後のサポートはありますか？ +
片道航空券の支給、空港送迎、さらに到着後6泊7日のホテルを提供（その間に住居探し）。初めての海外でも安心して移住を開始できます。
家族の帯同は可能ですか？ +
就労ビザ（EP）取得後にDependent Pass（配偶者・子ども）を申請できます。日本人学校や日本語医療もあり、帯同移住の方も多いです。

仕事内容・働き方について

どんな業務を担当しますか？ +
日本のお客様対応（電話・メール・チャット）。Eコマース、旅行、ホテル、金融、SNS、ITなど多岐。注文・予約・アカウント・技術サポートなど、課題解決を支援します。
勤務時間やシフトは？リモートは？ +
週5日・1日8〜9時間が基本。日本時間に合わせた早朝/夜間もあり、シフト手当対象。職種によりハイブリッド勤務可（詳細は面接で案内）。
キャリアパスはありますか？ +
実力重視の評価で、トレーナー / QA / チームリーダー / マネージャーなどへ昇進可能。別プロジェクトへの異動機会もあります。
研修や語学サポートは？（GoFluent等） +
入社時研修＋継続的なトレーニングを提供。英語力向上にはLanguage Bootcamp / GoFluentなどの学習支援が利用できます。

給与・福利厚生について

有給・休暇制度は？ +
初年度年14日（勤続で最⼤21日）。病気休暇14日、現地祝日＋一部日本祝日も休み。プロジェクトにより異なる場合はオファー時に明示します。
昇給・昇進の機会は？ +
年2回の評価で昇給のチャンス。パフォーマンスに応じて役職登用もあります（年齢や在籍年数ではなく実力を重視）。
医療保険などの福利厚生は？ +
医療保険を完備。日本語対応の医療機関も利用しやすく、自己負担を抑えられます。各種手当（シフト等）、表彰・イベントなどの制度もあります。
退職時の帰国費用は支給されますか？ +
契約満了または会社都合の退職時は帰国航空券支給。自己都合退職は自己負担です（詳細は雇用契約をご確認ください）。

生活・住まいについて

日本と比べた生活コストは？（例：KL） +
物価が安く、時間・お金・心に“ゆとり”が生まれます。例（東京 vs KL）：
項目日本（東京）マレーシア（KL）
家賃（1LDK）約12万円〜約5〜6万円（24hセキュリティ・プール・ジム付）
外食1,000円〜約300円〜
交通費月1万円以上数千円以内（Grab移動）
社員の声：カフェ巡りや外食、平日夜の余暇も楽しみやすくなった等の声が多いです。
住居はどうやって探しますか？相場は？ +
到着後のホテル滞在中に内見・契約。日本人スタッフがエージェント紹介やアドバイスを行います。
KL/ペナンの一般相場はRM1,500〜2,500（約5〜8万円）でプール・ジム付きが一般的。家具家電付きで即入居しやすい物件が多いです。
日本食や日用品は入手しやすい？ +
KLはドン・キホーテ / AEON / 伊勢丹、バンコクはフジスーパー等があり、日本食レストランも充実。食に困ることはほぼありません。
医療や治安は大丈夫？ +
日本語対応の医療機関が複数あり、医療水準も高いです。治安は比較的安定。基本的な防犯意識を持てば、女性の一人暮らしでも安心して生活できます。
休日や週末はどんな過ごし方ができますか？ +
ランニング・ジム・ヨガ、カフェ巡り、ナイトマーケット、ビーチ（日帰り/週末）などアクティブなオフが楽しめます。
旅行好きにも好立地（LCCで近隣国へ気軽に旅行 / KL・バンコクは多方面直行便）。

各都市（KL/ペナン/バンコク）

クアラルンプール（KL）の特色は？ +
都市の利便性と多文化。MRT/モノレール/Grabで移動快適。Mont Kiara・KLCC等の人気エリアは高級コンドが多く、設備充実で日本より家賃は安価。病院や日本人会、日本語学校も充実し初めての海外でも安心です。
ペナン（Penang）の特色は？ +
自然×歴史の落ち着いた島暮らし。海に近い眺望の良い高層コンドが多く、KLよりやや生活費が安い傾向。長期滞在にも人気で、家族帯同にも向いています。
バンコク（Bangkok）の特色は？ +
“東南アジアの東京”とも言える便利な大都市。BTS/MRT発達、家具付きコンド（プール/ジム）で家賃は東京の約半額感。日本語施設や医療が非常に充実し、親日的で働きやすい環境です。

会社・カルチャーについて

TPはどんな会社？認証・実績は？ +
1978年フランス創業、世界100カ国以上・従業員50万人以上のBPOリーダー。TPマレーシアはGreat Place to Work®認証を取得し、働きやすい職場環境が評価されています。多国籍・フラットな社風です。
社内カルチャーやイベントはありますか？ +
社内イベント、表彰、チームビルディングなど人とのつながりを大切にするカルチャー。国籍・年齢・性別を問わず尊重し合う文化で、価値観や視野が広がります。
採用チーム（TA）はどんなサポートをしてくれますか？ +
日本語対応のTAチームが応募〜入社まで丁寧に伴走。選考や現地生活の不安も日本語で相談OKです（例：Joseph/シニアマネージャーをはじめ、多国籍メンバーが在籍）。
